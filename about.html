1991: Tim Berners-Lee created the first web page, a simple text-based document without any styling. Early 1990s: As the web evolved, the need for styling became evident. Early attempts at styling involved using HTML tags like to control text appearance, leading to cumbersome and inconsistent designs across multiple pages. The Birth of CSS: 1994: HÃ¥kon Wium Lie, working at CERN, proposed the concept of Cascading HTML Style Sheets, later shortened to Cascading Style Sheets (CSS). This proposal introduced the revolutionary "cascading" principle, allowing styles to originate from multiple sources (author, user, browser) and be applied based on a set of rules. 1996: CSS Level 1 (CSS1) became a W3C Recommendation in December. 1998: CSS Level 2 (CSS2) was released, introducing more advanced features for positioning, media-specific styles, and accessibility. Late 1990s - Early 2000s: Browser support for CSS evolved, with early adopters like Internet Explorer 3 and Netscape Navigator 4 beginning to implement CSS features. This period also saw the rise of cross-browser compatibility challenges due to varying implementations. 2000s onwards: CSS development continued with CSS Level 3 (CSS3), which departed from monolithic specifications and was published as a collection of separate modules. This modular approach allowed for more flexible and incremental development of new features, such as advanced selectors, transitions, animations, and responsive design capabilities.
